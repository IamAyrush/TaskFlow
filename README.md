# TaskFlow — Task Management Application

A beautiful, frontend-only task management application built with React, TypeScript, and MSW for API mocking.

**Live Demo:** Deploy to Vercel/Netlify by connecting your GitHub repo.

---

## Features

- **Authentication** — Mocked login with JWT simulation, persisted via localStorage
- **Task CRUD** — Create, read, update, delete tasks with title, description, and status
- **Filtering** — Filter tasks by status (All / To Do / In Progress / Done)
- **Animated UI** — Canvas particle background, smooth transitions, black/yellow/white theme
- **Responsive** — Mobile-friendly layout with Tailwind CSS
- **Form Validation** — Formik + Yup for real-time validation

## How the Mocking Works

This app uses **Mock Service Worker (MSW)** to intercept HTTP requests in the browser and return fake responses — no real backend needed.

### Setup

1. **Service Worker** — `public/mockServiceWorker.js` (generated by MSW) intercepts network requests at the browser level
2. **Handlers** — `src/mocks/handlers.ts` defines request handlers for each endpoint
3. **In-memory store** — Tasks are stored in a module-level array, simulating a database
4. **Startup** — `src/main.tsx` starts the worker before rendering React

### Mock Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/login` | Validates credentials, returns fake JWT + user |
| `GET` | `/api/tasks` | Returns all tasks (requires Bearer token) |
| `POST` | `/api/tasks` | Creates a new task (requires Bearer token) |
| `PUT` | `/api/tasks/:id` | Updates a task by ID (requires Bearer token) |
| `DELETE` | `/api/tasks/:id` | Deletes a task by ID (requires Bearer token) |

### Test User

```
username: test

```

### Persistence

- **Auth token** is stored in `localStorage` under `task_manager_token`
- **User info** is stored in `localStorage` under `task_manager_user`
- Tasks are reset to initial seed data on page refresh (MSW in-memory store resets)

## Project Structure

```

├── __mocks__/          # Jest file stubs
├── __tests__/          # Unit & integration tests
├── components/
│   ├── layout/         # Header, PrivateRoute
│   ├── tasks/          # TaskCard, TaskForm, TaskModal, TaskList
│   └── ui/             # ParticleBackground, LoadingSpinner, EmptyState
├── hooks/              # useAppDispatch, useAppSelector

│   ├── browser.ts      # MSW browser worker setup
│   ├── data.ts         # Seed data and mock credentials
│   ├── handlers.ts     # MSW request handlers
```
